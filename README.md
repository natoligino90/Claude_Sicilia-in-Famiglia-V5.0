## 🚀 Deploy su GitHub Pages

1. Crea un repository GitHub (pubblico)
2. Carica tutti i file nella cartella root
3. Vai in **Settings → Pages → Source: main / root**
4. Il sito sarà live su: `https://<username>.github.io/<repo>/`
